### npm升级package.json依赖包到最新版本号
#### 安装
```bash 
npm install -g npm-check-updates
```
#### 使用
##### 检查package.json中dependencies的最新版本
```bash
ncu
```
##### 更新dependencies到新版本
```bash
ncu -u
```
##### 更新全部到最新版本：
```bash
ncu -a
```
